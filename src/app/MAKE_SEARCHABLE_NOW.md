## ⚡ **STEP 1: DEPLOY YOUR SITE (5 MINUTES)**


```
https://dh-nexus.vercel.app
```

✅ **DONE! Site is online!**

---

## 🔍 **STEP 2: SUBMIT TO GOOGLE (5 MINUTES)**

### **A. Go to Google Search Console:**
👉 https://search.google.com/search-console

### **B. Add Your Site:**
1. Click **"Add Property"**
2. Select **"URL prefix"**
3. Enter: `https://dh-nexus.vercel.app`
4. Click **"Continue"**

### **C. Verify Ownership:**

**Option 1 - HTML Tag (Easiest):**
1. Google shows you a meta tag like:
   ```html
   <meta name="google-site-verification" content="abc123...">
   ```
2. **ALREADY DONE!** The meta tag section is ready in `/index.html`
3. Just add your verification code to the index.html file
4. Redeploy: `vercel --prod`
5. Click **"Verify"** in Google Search Console
6. ✅ **Verified!**

**Option 2 - HTML File:**
1. Download verification file (e.g., `google123abc.html`)
2. Put it in `/public` folder
3. Redeploy: `vercel --prod`
4. Click **"Verify"**
5. ✅ **Verified!**

### **D. Submit Sitemap:**
1. In Google Search Console, click **"Sitemaps"** (left menu)
2. Enter: `sitemap.xml`
3. Click **"Submit"**
4. ✅ **Sitemap submitted!**

You'll see:
```
✅ Success
Discovered URLs: 24
```
