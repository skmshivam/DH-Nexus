---

### **STEP 2: Google Search Console (5 minutes)**

#### **A. Create Account:**
1. Go to: https://search.google.com/search-console
2. Click "Start now"
3. Sign in with Google account

#### **B. Add Your Property:**
1. Click "Add Property"
2. Choose "URL prefix"
3. Enter: `https://dh-nexus.vercel.app`
   - OR if you have custom domain: `https://dhnexus.com`
4. Click "Continue"

#### **C. Verify Ownership:**

**Option 1: HTML File Upload (Easiest)**
1. Download verification file (e.g., `google1234567890abcdef.html`)
2. Create file in `/public` folder:
   ```bash
   # Create verification file
   touch public/google1234567890abcdef.html
   # Add content Google provides
   echo "google-site-verification: google1234567890abcdef.html" > public/google1234567890abcdef.html
   ```
3. Deploy again:
   ```bash
   vercel --prod
   ```
4. Go back to Search Console
5. Click "Verify"
6. ✅ **Verified!**

**Option 2: HTML Tag (Alternative)**
1. Google gives you a meta tag like:
   ```html
   <meta name="google-site-verification" content="1234567890abcdef" />
   ```
2. Already added to `/index.html`! ✅
3. Deploy and verify

#### **D. Submit Sitemap:**
1. In Search Console, click "Sitemaps" (left sidebar)
2. Enter: `sitemap.xml`
3. Click "Submit"
4. ✅ **Sitemap submitted!**

Status will show:
```
✅ Success
Discovered URLs: 24
```
