## 🚀 **Best Practices for External Linking**

### 1. **Always use HTTPS**
```
✅ https://dhnexus.com/talent
❌ http://dhnexus.com/talent
```

### 2. **Use UTM parameters for tracking**
```
https://dhnexus.com/join-talent?utm_source=website&utm_medium=banner&utm_campaign=student_drive
```

### 3. **Use descriptive anchor text**
```html
✅ <a href="https://dhnexus.com/for-recruiters">Hire Digital Humanities Professionals</a>
❌ <a href="https://dhnexus.com/for-recruiters">Click Here</a>
```

### 4. **Open in new tab for external sites**
```html
<a href="https://dhnexus.com/talent" target="_blank" rel="noopener noreferrer">
  Browse DH Talent
</a>
```
