## 🔍 **STEP 3: ADD GOOGLE VERIFICATION**

### **A. Go to Google Search Console:**

1. Open: **https://search.google.com/search-console**
2. Click **"Start now"**
3. Sign in with Google

### **B. Add Your Site:**

1. Click **"Add property"** (top left dropdown)
2. Choose **"URL prefix"** (right option)
3. Enter: `https://dh-nexus-xyz123.vercel.app`
   (use YOUR actual URL from Step 1)
4. Click **"Continue"**

### **C. Get Verification Code:**

1. Google shows verification methods
2. Click **"HTML tag"**
3. You'll see code like:
   ```html
   <meta name="google-site-verification" content="ABC123xyz789" />
   ```
4. **COPY the content value:** `ABC123xyz789`
   (Your actual code will be different!)

### **D. Add Code to Your Site:**

1. **Open** `/index.html` in your code editor
2. **Find line 18** (says "PASTE_YOUR_CODE_HERE")
3. **Replace** `PASTE_YOUR_CODE_HERE` with your actual code
4. **Example:**
   ```html
   <!-- Before: -->
   <meta name="google-site-verification" content="PASTE_YOUR_CODE_HERE" />
   
   <!-- After: -->
   <meta name="google-site-verification" content="ABC123xyz789" />
   ```
5. **Save the file**

### **E. Redeploy:**

```bash
# In terminal, run:
vercel --prod
```

## 🚨 **TROUBLESHOOTING**

### **Problem: Vercel command not found**
```bash
# Try with npx:
npx vercel
```

### **Problem: Verification failed**
- Make sure code is in `/index.html` line 18
- Redeploy: `vercel --prod`
- Wait 5 minutes
- Try verify again

### **Problem: Sitemap not found**
- Check: `https://your-url.vercel.app/sitemap.xml`
- If 404, the `/public/sitemap.xml` file exists
- Redeploy: `vercel --prod`

### **Problem: Not appearing on Google after 7 days**
- Check Google Search Console for errors
- Re-submit sitemap
- Request indexing manually
- Make sure robots.txt allows indexing
