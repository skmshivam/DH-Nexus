## STEP 1: DEPLOY TO VERCEL (5 MINUTES)

### **Option A: Deploy with GitHub (Recommended)**

```bash
# 1. Create GitHub repository
# Go to: https://github.com/new
# Name: dh-nexus
# Click "Create repository"

# 2. Push your code
git init
git add .
git commit -m "DH Nexus - Production Ready"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/dh-nexus.git
git push -u origin main

# 3. Deploy to Vercel
# Go to: https://vercel.com/new
# Click "Import Git Repository"
# Select your dh-nexus repository
# Click "Deploy"
# DONE! ✅
```

**Your site will be live at:**
```
https://dh-nexus.vercel.app
```

### **Option B: Deploy with Vercel CLI (Faster)**

```bash
# 1. Install Vercel CLI
npm install -g vercel

# 2. Login to Vercel
vercel login

# 3. Deploy
vercel

# Follow prompts:
# - Set up and deploy? YES
# - Which scope? [your account]
# - Link to existing project? NO
# - What's your project's name? dh-nexus
# - In which directory is your code? ./
# - Want to override settings? NO

# 4. Deploy to production
vercel --prod

# DONE! Your site is LIVE! ✅
```
